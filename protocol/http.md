# HTTP
  - `OSI 7계층` 상 7번째 계층인 `Application Layer` 속합니다.
  - Hyper Text Transfer Protocol 의 약자로 원래는 HTML 을 주고 받기 위함이 였지만, 현재는 Json, Image 등 다양한 데이터를 주고 받는 데 쓰인다. 
  - 대부분 `TCP` 를 이용한다.
  - Client 요청에 따라 데이터를 반환하는 단방향 통신.
