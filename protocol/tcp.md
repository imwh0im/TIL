# TCP
  - Transmission Control Protocol 의 약자이다.
  - `OSI 7계층` 상 4번째 계층인 `Transport Layer` 속합니다.
  - 연결 지향적 프로토콜이다.
    - Client 와 Server 가 연결 된 후에 통신하는 프로토콜을 뜻합니다. 
    - 그렇기에 양방향 통신을 지원합니다.
  - TCP 는 포트 번호를 통해 애플리케이션 계층과의 연결을 식별합니다.
  - 데이터의 누락이나 패킷의 순서 뒤바뀜 등 데이터를 검사하는 기능이 있습니다.
    - 데이터 전송 순서를 보장합니다.
